Audit Request Procedure



Audit Request Confluence: Click Here

Procedure:

{% load static %}




Login into the Audit Request Portal by entering your username, password, passcode.
Once logged in, you will be redirected to the home page of the portal.






Navigate to the Credentials tab from the navigation bar. You will be directed to the Server login page. Please enter the Server Username and Password for which you wish to generate the Audit Request. Once entering please save the credentials using the "Save" button.

{% load static %}
{% load static %}

After clicking the save button, use the navigation bar and click on the Jira tab. You will be redirected to the Jira tab.



Enter the appropriate Jira TicketId and click on "Fetch".

{% load static %}
{% load static %}



The Description of the above entered Jira ticket will be fetched as shown alongside.



Please enter your Server name on which the Audit Request Process should be performed.


Use the Checkbox and check the necessary commands to be included.



Any other commands that are needed to be executed can be entered into the textarea available(on individual lines).


Once done click the "Save" button and follow it up by clicking the "Execute" button.

{% load static %}
{% load static %}





Re-enter the TicketId and click on the "Upload" button. Your files will be uploaded to the respective Ticket on AthenaJira.






Enter your ServiceNow Username, Password and Sys Id and click on "Upload".Your files will be uploaded to the respective Incident on Athena ServiceNow.
