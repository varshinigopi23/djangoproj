Audit Request

A cloud audit is a periodic examination that an organization does to assess and document its cloud vendor performance. The goal of such an audit is to see how well a cloud vendor is doing in meeting a set of established controls and best practices.

The Cloud Infrastructure Engineering(CIE) - Production provides audit documents, guidance and controls that an IT organization can use to examine its cloud vendors. CIE resources are considered the primary audit tools to perform and optimize a comprehensive cloud audit.
